API Reference¶

This section documents the public API of the svdep package.

Main Classes¶

TaskBuildFileCollection¶

class TaskBuildFileCollection(root_paths, incdirs=None)¶

Builds a file collection by scanning root files and their includes.

Parameters:

  • root_paths (List[str]) – List of paths to root SystemVerilog files to scan.

  • incdirs (List[str], optional) – List of include directories to search for included files.

build()¶

Scan all root files and their includes, building a complete dependency tree.

Returns:

A FileCollection containing all discovered files and their dependencies.

Return type:

FileCollection

Example:

from svdep import TaskBuildFileCollection

task = TaskBuildFileCollection(
    root_paths=['top.sv', 'dut.sv'],
    incdirs=['include/', 'rtl/']
)
collection = task.build()

TaskCheckUpToDate¶

class TaskCheckUpToDate(root_files, incdirs=None)¶

Checks whether files in a collection are up-to-date relative to a timestamp.

Parameters:

  • root_files (List[str]) – List of paths to root SystemVerilog files.

  • incdirs (List[str], optional) – List of include directories.

check(info, timestamp)¶

Check if all files in the collection are up-to-date.

Parameters:

  • info (FileCollection) – The file collection to check.

  • timestamp (float) – Reference timestamp (e.g., last build time).

Returns:

True if all files are up-to-date (not modified since timestamp), False otherwise.

Return type:

bool

Example:

from svdep import TaskCheckUpToDate, FileCollection
import os

checker = TaskCheckUpToDate(
    root_files=['top.sv', 'dut.sv'],
    incdirs=['include/']
)

# Load existing collection
collection = FileCollection.from_dict(saved_data)

# Check against build output timestamp
build_time = os.path.getmtime('output.bin')
is_current = checker.check(collection, build_time)
